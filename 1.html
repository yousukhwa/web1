HTML이란 무엇인가?

Hypertext Markup language(HTML) is the standard markup language for creating web pages and web application. WEB browser received HTML documents from a web server or from local storage.

HTML elements are the building blocks of HTML pages. Whith HTML constructs, images and other objects, such as interactive forms, may be embedded int the rendered page.
